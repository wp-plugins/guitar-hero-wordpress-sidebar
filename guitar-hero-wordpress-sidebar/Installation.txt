This section describes how to install the plugin and get it working.

1. Download the archive and unzip it

2. Upload the files to the /wp-content/plugins/guitar-hero directory

3. Activate the plugin through the 'Plugins' menu in WordPress

4. Place <?php if (function_exists("gh_stats")) { gh_stats(); } ?> in your templates where you want to display the statistics
